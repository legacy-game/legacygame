# legacy

*legacy* is a persistent civilization sandbox / cozy life sim with a civic meta-layer.

The current milestone is the first playable shared-world prototype: a small playable block of Veyne where citizens, property ownership, jobs, money, time, schedules, persistence, and history all exist in one shared world state.

## Development

- Engine: Unity 6.0.1f1
- Render pipeline: URP 2D
- Visual target: Stardew-style 2.5D pixel art
- Source control: Git + Git LFS

Unity project files should live at the repository root, with authored project files under `Assets/_Project/` once the Unity project structure is created.
